Test Instructions:

This test demonstrates the image preview functionality that has been added to the Gmail Chrome extension. The extension now supports both PDF and image previews using the same workflow.

  • PDF Test: Creates a test PDF document and displays it in an iframe
  • Image Test: Creates a test image using HTML5 Canvas and displays it as an img element

Click a test button above to see the preview functionality

Implementation Details:

  • File Type Detection: The system detects file types based on filename extensions (.pdf, .png, .jpg, .jpeg, .gif)
  • Dynamic Button Text: Buttons show "TEST PDF" or "TEST IMAGE" based on the file type
  • Blob Creation: Different test blobs are created for PDFs (string-based) and images (canvas-based)
  • Display Logic: PDFs use iframe elements, images use img elements with object-fit: contain
  • Same Workflow: Both file types use the same preview workflow and caching system
